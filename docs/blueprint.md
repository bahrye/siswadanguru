# **App Name**: EduCloud Manager

## Core Features:

- Multi-School Dashboard: Display an overview of all schools, with summaries of student and teacher data.
- Admin Login: Secure login system for administrators to manage school, student, and teacher data.
- School Management (CRUD): Enable admins to create, read, update, and delete school records.
- Student Data Management (CRUD): Enable admins to create, read, update, and delete student records for each school.
- Teacher Data Management (CRUD): Enable admins to create, read, update, and delete teacher records for each school.
- Data Sync with Firestore: Real-time synchronization of all data with Firestore for persistence and reliability.
- AI-Powered Data Validator: Suggest automatic anomaly detection when updating records.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism, while still feeling modern.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop for the data.
- Accent color: Orange (#FF9800) to highlight important actions and calls to action, providing visual contrast.
- Body font: 'Inter', a grotesque-style sans-serif font providing a modern, machined and neutral look. It ensures readability for all text.
- Headline font: 'Space Grotesk', a proportional sans-serif that gives a techy, scientific feel. This font choice is suitable for headlines
- Use a consistent set of minimalist icons for navigation and data representation. The set should be appropriate to educational contexts.
- Implement a responsive sidebar for navigation and ensure the application is fully responsive on mobile devices.
- Incorporate subtle animations for loading states and transitions to enhance the user experience.
